# Weather Application

🌐 Live Demo: https://prasad-gorige.github.io/weather/

## 📌 Description
This is a responsive Weather Application that provides real-time weather information for any city using API integration.

## 🚀 Features
- Search weather by city
- Displays temperature, humidity, and weather conditions
- Real-time data using API
- Responsive design (mobile & desktop)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
